# Movie Recommender System (SVD vs PMF)

Advanced movie recommendation engine for a streaming platform startup. 
Benchmarking Singular Value Decomposition (SVD) against Probabilistic Matrix Factorization (PMF).

## Setup Instructions
1. Activate virtual environment: `source venv/bin/activate`
2. Install dependencies: `pip install -r requirements.txt`
3. Run dashboard: `streamlit run app.py`# matrix-factorization
